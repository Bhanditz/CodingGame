Welcome to RoboBrains game

You are controlling a robo that has fallen down on a mysterious planet. You can help him move by implementing a program in his brain.

The robo_action function

That is the main function of the robo brain. You can return either game.Jump() or game.Forward() for example to make robo jump or move forward. You may investigate your robo argument. It provides access to all robo subsystems: gps, sensors, laser.

The robo argument has the following functions:
gps.x
Robo horizontal position
gps.y
Robo vertincal position
sensors.is_barrier_near()
Function that returns True if there is a barrier in front of the robo. You can use it to pass obstacles
